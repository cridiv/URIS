# Planner Agent

Plans data quality improvements based on dataset evaluation and user goals.

## Functionality

- Receives evaluation summary and user AI/ML objective
- Generates ordered task list for data preparation
- Estimates baseline ADFI (AI Data Fitness Index) scores
- Defines constraints and revision triggers

## Key Output

- Objective statement (data readiness goal)
- Prioritized task sequence for evaluation, compliance, synthesis, validation
- Risk tolerance level and constraints
- Baseline quality estimates (completeness, balance, compliance risk)

## Scope

**Does NOT** recommend ML models, hyperparameters, or feature engineering—focuses solely on data quality and preparation.
